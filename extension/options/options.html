🛡️ Social Monitor

Cấu hình extension để kiểm tra bản quyền ngay trên YouTube.

Kết nối với ứng dụng

Địa chỉ nơi bạn đã deploy app (hoặc http://localhost:3000 khi dev).

Lấy key tại Settings → Extension Key trong ứng dụng.

Chưa kiểm tra kết nối
Hướng dẫn
  1. Deploy hoặc chạy local ứng dụng Social Monitor.
  2. Đăng nhập vào app → vào Settings → Extension Key.
  3. Bấm Tạo API Key, copy key.
  4. Dán URL app và key vào ô trên, bấm Lưu.
  5. Mở bất kỳ video YouTube — panel sẽ hiện ở sidebar phải.
