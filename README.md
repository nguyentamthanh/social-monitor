# Social Monitor

Keyword monitoring dashboard for Facebook, Google, YouTube, and TikTok.

## Setup

1. Copy `.env.local.example` to `.env.local` and add your MongoDB Atlas URI
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:3000

## Deploy to Vercel

1. Push to GitHub
2. Import to Vercel
3. Add environment variables (MONGODB_URI, NEXTAUTH_SECRET, NEXTAUTH_URL)
4. Deploy
